# Agent Orchestration

A security first platform for governed, identity aware AI agent execution. It is a distinct Goji Labs product and does not depend on the DeFi product family.

## What it is

A centralized control plane that orchestrates autonomous AI agents with the governance, identity enforcement, and audit that regulated enterprises require, closing the gap between agent capability and operational trust.

## Status

The current artifacts are imported decks and concept designs (`concept-design.pdf`, `v2.pdf`). They will be restructured into the standard Markdown document set when this line is picked up for active work.

## Note

This platform, together with Corralign for identity, is the natural execution and governance substrate for running the Dynamic Liquidity engine and the Community Platform as a multi tenant service. That composition is a strategic option, recorded here so the dependency is visible, not a current coupling.
