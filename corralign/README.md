# Corralign

An identity and LLM control model. Corralign is a distinct Goji Labs product and does not depend on the DeFi product family.

## Status

The current artifact is an imported concept design (`concept-design.pdf`). It will be restructured into the standard Markdown document set when this line is picked up for active work.

## Note

Corralign is a strong candidate to provide the identity and access control layer for the Agent Orchestration platform, and for KYC and governed access in the Community Platform. That composition is a strategic option, not a dependency. The product stands on its own.
